# 📚 Lógica de Programação

Repositório com exercícios desenvolvidos na disciplina de Lógica de Programação.

## 🎯 Objetivo

Praticar raciocínio lógico e aplicar os fundamentos da programação através da resolução de problemas utilizando diferentes linguagens.

## 🛠 Linguagens utilizadas

- Python
- C++

## 📂 Organização

Os exercícios estão organizados em pastas conforme as listas e atividades propostas durante a disciplina.

## 🚀 Sobre

Este repositório faz parte do meu processo de aprendizado na área de tecnologia, contribuindo para o desenvolvimento de uma base sólida em programação e resolução de problemas.
